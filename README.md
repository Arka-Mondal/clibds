# clibds -- A Data Structure library written in C

#### Data Structures:
- [x] Linked list
- [x] Stack
- [x] Queue
- [x] Priority Queue
- [ ] Vector (A contiguous growable array)
- [ ] Hash Table
- [ ] Hash Map
- [ ] AVL Tree
- [ ] Red Black Tree
- [ ] Graph

Every Data Structure has its own directory and contains a **example.c** describing
the usage procedure.
